Board game networks

Board game network dataset.

Information, documentation, source, metadata, license, credits, etc. are available on GitHub.. Or, you can clone this website that includes the processed data (graph files you can import into other tools): git clone https://github.com/rkdarst/board-game-networks --branch gh-pages

This data is under production and is not complete or well documented yet.

You can download metadata at data.json.
